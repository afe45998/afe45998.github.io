2.关键字实参：其中每个实参都由变量名和值组成

3.还可以使用列表和字典

8.2.1 位置实参

在调用函数过程中，Python必须将函数调用中的每个实参都关联到函数定义中的一个形参。为此，最简单的关联方式是基于实参的顺序————位置实参

8.2.2 关键字实参

关键字实参时传递给函数的名称-值对。关键字实参使我们无需考虑函数调用中的实参顺序，还清楚地指出了函数调用中各个值的用途

关键字实参的顺序无关紧要，弹药准确地指定函数定义中的形参名
